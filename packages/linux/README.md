# NADE - Linux Package

**Version 1.0.0**

Cross-platform audio encryption system - Linux build

## Quick Start

```bash
# Install dependencies
pip3 install --user -r requirements.txt

# Run NADE
./run_nade.sh
```

## Manual Installation

```bash
# Install Python packages
pip3 install --user sounddevice numpy

# Run executable
./bin/NADE
```
